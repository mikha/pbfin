программа производит сбор финансовой статистики по всем клубам ПБ-Лиги и дополнительно группирует данные по федерациям/странам. поскольку в качестве источника используется [дополнительная статистика](http://pbliga.com/stat_ext.php), доступ к которой возможен только для авторизованных ВИП-пользователей, при запуске в качестве первого параметра необходимо указать ИД сессии:
    java com.dewdrop.other.money.Aggregator a1b2cdef34g56789hijk0l1m23

для нахождения вашей текущей ИД сессии необходимо немного порыться в браузере и вытянуть значение куки с именем PHPSESSID для сайта pbliga.com
